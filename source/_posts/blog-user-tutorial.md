# 提交一篇post基本操作
- 进入d://Blog/sourceFile/

- 新建post:

  hexo new post PathName

- 编辑文本：使用[Typora](https://www.typora.io/)编辑文本

- hexo g :编译

- hexo s：本地展示

- hexo d：推送到线上

- 查看生成结果（需要等待一段时间）
